You are a senior front-end designer and developer.
Create a highly attractive, modern, responsive website for a photography and studio business that offers:

Passport size photos

Personal / portrait shoots

Wedding & pre-wedding shoots

Birthday and event shoots

General studio photography services

Assume the studio is local (Indian city) and wants to impress walk-in and online clients.

A) Overall goals

The site should look like a professional photography studio: visual, elegant, and easy to use.

Focus on beautiful galleries, clear services, and strong calls to action (call / WhatsApp / book appointment).

Design must work great on mobile as well as desktop.

B) Structure & pages (single-page with sections)
Build a single-page scrolling site with a sticky navbar linking to these sections:

Hero (Home)

Services

Portfolio / Gallery

Packages & Pricing

About the Studio

Testimonials

FAQ

Contact & Location

C) Hero section

Full-width hero with a large background photo (blurred or dark overlay) related to studio/portrait photography.

Prominent studio name (you can use “Adhithyan Studio & Photography” as a placeholder).

Subheading: mention passport photos, personal portraits, wedding, pre-wedding, birthday, and more.

Primary CTA buttons:

“Book a Photoshoot”

“Call / WhatsApp Now” (use placeholder phone number).

Short line showing location and tagline like: “Professional photography studio for all your special moments.”

D) Services section
Create cards or tiles for main service categories:

Passport & ID Photos

Personal / Portrait Shoots

Wedding Photography

Pre-Wedding Shoots

Birthday & Event Coverage

Family / Group Photos

Studio Product Shoots (optional)

Each card should have:

Icon or small image

Service name

2–3 line description

Optional “View Samples” link that scrolls to relevant gallery area.

E) Portfolio / Gallery section

A responsive gallery with filters/tabs for:

All

Passport & Portrait

Wedding & Pre-Wedding

Birthday & Events

Use a grid layout (2–4 columns depending on screen size).

On click, open a lightbox/modal with a larger image and caption.

Use placeholder images or sample URLs, but structure the HTML so images are easy to replace.

F) Packages & Pricing section
Show clearly defined packages:

Passport Photos Package (quick service, instant prints, digital copy option).

Portrait Session Package (duration, edited photos count).

Wedding / Pre-Wedding Packages (basic, standard, premium – short descriptions).

Birthday / Event Coverage Package (hours of coverage, number of photos).

Use pricing cards with:

Package name

Short description

Starting price placeholder (e.g., “Starts from ₹X,XXX”).

Bullet list of what’s included.

A “Book Now” or “Enquire” button.

G) About the Studio section
Include:

Short story about the studio, years of experience, style (candid, traditional, creative).

Mention that they handle everything from passport photos to full wedding coverage.

Small highlights row:

“High-quality prints & albums”

“Quick passport photo delivery”

“Professional lighting & studio setup”

“Friendly and experienced photographers”

H) Testimonials section
Add 3–4 sample testimonials:

One for passport photos (quick and accurate).

One for wedding photography (captured all moments).

One for birthday / event.

Use card layout with client name, occasion, short feedback, and a subtle quote icon.

I) FAQ section
Add common questions like:

How long does it take to get passport photos?

Do you provide both printed and digital copies?

How far in advance should we book for weddings?

Do you travel to outside locations for shoots?

Use an accordion-style FAQ (can be pure CSS or minimal JS).

J) Contact & Location section

Contact form fields: Name, Email, Phone, Type of Shoot (dropdown), Date, Message.

Show phone number, WhatsApp, email, and studio address clearly.

Include a small embedded map area (use a placeholder div styled like a map).

Buttons: “Call Now”, “Get Directions”, “WhatsApp Us”.

K) Design & visual style

Use a dark-on-light or light-on-dark theme that makes photos pop.

High emphasis on large, high-quality images, with good spacing and minimal clutter.

Use a calm, elegant color palette (e.g., dark charcoal + gold/amber accent, or deep navy + soft beige).

Choose a modern sans-serif font for text, optionally a stylish serif for headings.

L) Interactions & “attractive” features

Smooth scrolling for menu links.

Hover effects for service cards and gallery images (scale, shadow, overlay).

Lightbox for gallery images.

Subtle section reveal animations (e.g., fade-in on scroll using basic CSS/JS, but keep it simple).

Sticky header that becomes slightly smaller or adds a shadow when scrolling.

M) Technical details

Use pure HTML, CSS, and vanilla JavaScript only (no frameworks like Bootstrap/Tailwind).

Make it fully responsive: test breakpoints for mobile, tablet, and desktop.

Optimize image usage with CSS so layout remains fast and smooth even with many photos.

Code should be clean, semantic, and beginner-friendly, with comments separating sections (<!-- Hero -->, <!-- Services -->, etc.).

N) Output format

Provide a single HTML file, with a <style> block in <head> and <script> at the bottom (if needed).

Fill all text with realistic, professional content for a photography studio (no lorem ipsum).

Use placeholder images/links that I can easily replace later.

Now generate the full HTML, CSS, and minimal JS for this photography studio website according to all the instructions above.
